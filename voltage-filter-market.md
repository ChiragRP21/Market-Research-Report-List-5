<p><ul><li>Low Pass Filter</li><li>High Pass Filter</li><li>Band-pass filter</li><li>Band Rejection Filter</li></ul></p>
<p><p>Voltage filters are used to control the flow of electrical signals in a circuit by allowing some frequencies to pass through while blocking others. The four main types of voltage filters are Low Pass Filter, which allows low frequencies to pass through, High Pass Filter, which allows high frequencies to pass through, Band-pass filter, which allows a specific range of frequencies to pass through, and Band Rejection Filter, which blocks a specific range of frequencies while allowing others to pass through. Each type of filter serves a specific purpose in regulating electrical signals.</p></p>
